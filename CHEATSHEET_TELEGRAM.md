# Bot Toyota iQ — Cheatsheet pour Surfeur d'Argent 🤖👀

### Pour commencer
1. Ouvrez ce lien dans Telegram : https://t.me/Tarek_IQ20_bot
2. Tapez `/start` et appuyez sur Envoyer

C'est tout ! Vous etes connecte.

### Commandes
Tapez le mot dans le chat et appuyez sur Envoyer (comme un message normal) :

| Vous tapez | Ce qui se passe |
|------------|-----------------|
| `chercher` | Lance une recherche sur LeBonCoin + AutoScout24. Vous recevez les 5 meilleures annonces avec photos |
| `liste` | Renvoie les dernieres annonces trouvees |
| `details 3` | Voir toutes les photos et la description complete de l'annonce #3 |
| `approuver 1,3` | Garder les annonces #1 et #3 pour negociation |
| `rejeter 2` | Supprimer l'annonce #2 de la liste |
| `effacer` | Nettoyer les messages du bot dans le chat |
| `intervalle 4h` | Programmer une recherche automatique (voir ci-dessous) |
| `statut` | Voir ou en est la recherche |

### Programmer la recherche automatique
Tapez `intervalle` suivi du delai souhaite :

- `intervalle 2h` → recherche toutes les 2 heures
- `intervalle 1j` → une fois par jour
- `intervalle 1s` → une fois par semaine

Minimum : 1 heure | Maximum : 1 semaine

### Ce que vous recevez par annonce
- Photo principale de la voiture
- Prix, annee, kilometrage
- Ville et distance depuis Orly
- Nom du vendeur (+ telephone si disponible)
- Points forts et points faibles
- Lien direct vers l'annonce

### Astuce
Pas besoin du `/` devant les commandes — tapez juste le mot et envoyez !
